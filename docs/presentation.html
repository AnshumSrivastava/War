Architecture Breakdown

Strict Separation of Concerns (SoC) following V3 standards:

  • State Layer: Immutable world snapshot & in-memory data.
  • Simulation Layer: Deterministic math & AI loops.
  • Presentation Layer: Reactive PyQt5/Web frontends.

The Simulation Heartbeat

Recursive flow of the ActionModel:

  • Sense: Agents query surrounding hexes.
  • Think: AI processes Q-Table rewards.
  • Act: Queue movement or fire orders.
  • Resolve: Deterministic casualty calc.

Code Integrity


# engine/ai/reward.py
def _get_goal_reward(entity, zone_hexes, step, max_steps):
    """Reward for reaching the objective zone, decaying over time."""
    multiplier = max(0.1, 1.0 - (step / max_steps))
    return REWARD_GOAL_REACHED * multiplier

# engine/ai/q_table.py  
def update(s, a, r, s_next):
    Q[s][a] += alpha * (r + gamma * max(Q[s_next]) - Q[s][a])

Getting Started

To launch the desktop interface:

python main.py

To run the web-based API & interface:

python web_ui/app.py

Q-tables are stored in-memory during training and saved to disk as .npy files.
